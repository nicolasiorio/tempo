# 🎵 tempo

Personal workspace for the rhythm of the working day — daily routines, current work items, and notes in one local-only dashboard.

Built as a sibling to [kit-db](https://github.com/nicolasiorio/kit-db): same stack (SvelteKit 2 / Svelte 5 / Tailwind v4 / Bun / bun:sqlite), same local-only posture, installed as a PWA on `http://127.0.0.1:3001`.
